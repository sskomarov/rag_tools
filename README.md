# RAG Tools
Набор инструментов и функций для использования при построения RAG-систем (паттерн Classic RAG).

## Описание
Представлен набор инструментов для использования при построения RAG-систем (паттерн Classic RAG) с помощью распространенных фреймворков (langchain, chonkie), оценки ретривера и генерации (метрики на основе n-грамм, семантические метрики).

## Документация
Пользовательскую документацию можно получить по [этой ссылке](./docs/ru/index.md).

## Что доступно в RAG Tools?
| Функционал                    | Описание                                                        |
|-------------------------------|-----------------------------------------------------------------|
| bm25                          | Алгоритм полнотекстового поиска BM25                            |
| embeddings                    | Векторный поиск с помощью различных моделей                     |
| rag-metrics                   | Метрики качества ретривера в RAG                                |
| llm-metrics                   | Метрики качества llm в RAG                                      |
